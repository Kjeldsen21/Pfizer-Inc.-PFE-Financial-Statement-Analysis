# Pfizer Inc. (PFE) Financial Statement Analysis
## Description

An academic project featuring a financial analysis of Pfizer Inc. built from the company's 2022 SEC 10-K filings. Raw GAAP financials extracted, structured, and analysed across six worksheets covering cash flow,leverage ratios, and working capital efficiency.
 
The project weighed 30% of the final grade. Received an A+.


## Sheets Included

| Sheet | Content |
|---|---|
| **Sheet 1:** Corporate Profile & Data | Consolidated Balance Sheet + Income Statement (FY2020–2022) |
| **Sheet 2:** Cash Flow Statement | Simplified operating, investing & financing activities with commentary |
| **Sheet 3:** Ratios & Common Size Balance Sheet | 10 financial ratios + vertical/horizontal % analysis |
| **Sheet 4:** Income Statement | Simplified P&L with YoY % change analysis |
| **Sheet 5:** DOL | Degree of Operating, Financial & Combined Leverage (DOL / DFL / DCL) |
| **Sheet 6:** CCC | Cash Conversion Cycle (DIO, DSO, DPO for FY2021–2022) |


## Key Numbers
